# SmartPriceTracker

An intelligent e-commerce price tracking application that monitors product prices across multiple platforms and helps users find the best deals.

## Features

- Track prices across multiple e-commerce platforms
- Get price change notifications
- Compare prices for similar products
- View price history
- Find similar products
- User-friendly web interface

## Requirements

- Python 3.x
- Flask
- SQLite

## Installation

1. Clone the repository:
```bash
git clone https://github.com/mallikarjun3065/Ecommerce-price-tracker.git
cd SmartPriceTracker
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python app.py
```

4. Open your browser and navigate to `http://localhost:5000`

## Project Structure

- `app.py` - Main Flask application
- `db.py` - Database management
- `scraper.py` - Web scraper for price extraction
- `tracker.py` - Price tracking logic
- `templates/` - HTML templates for the web interface
- `requirements.txt` - Python dependencies

## Usage

1. Add products you want to track
2. View price history and trends
3. Compare prices with similar products
4. Get notifications when prices drop
